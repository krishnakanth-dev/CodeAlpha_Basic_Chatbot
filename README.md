# 🤖 CodeAlpha Basic Chatbot

A simple rule-based chatbot developed using Python as part of the CodeAlpha Python Programming Internship.

## Features

- Greeting responses
- Date and Time display
- Basic Calculator
- Help menu
- Exit command
- Input validation

## Technologies Used

- Python
- Loops
- Conditional Statements
- Functions
- Exception Handling
- Datetime Module

## Commands

- hello
- how are you
- time
- date
- calculate
- help
- bye

## How to Run

```bash
python chatbot.py
```
